📖 使用说明

如何使用 MCP HTTP 服务

1. 启动服务器:

python mcp_server_http.py

2. 使用 curl 测试:

curl -X POST http://127.0.0.1:8080/message \ -H "Content-Type: application/json" \ -d '{"method": "add", "params": {"a": 5, "b": 3}}'

3. 在浏览器中访问:

http://127.0.0.1:8080

4. 连接到 MCP 客户端:

// 使用 HTTP 传输协议连接 const mcpClient = new MCPClient({ transport: 'streamable-http', url: 'http://127.0.0.1:8080' });
